가람 - "한글은 디자인이다"
=============
![Flake Banner](https://i.imgur.com/nJPjGjD.png)
가람(Garam)은 Kotlin으로 만들어진 인터프리터 언어입니다.   
한글날 기념으로 개발되었으며, 튜링 완전한 언어입니다.   

명령어
-------------
가람의 명령어는 8개로 구성되어 있습니다.
   
```ㅗ``` : 포인터 증가
```ㅜ``` : 포인터 감소
```ㅏ``` : 값 증가
```ㅓ``` : 값 감소
```ㅇ``` : 값 출력하기
```ㅁ``` : 값 입력받기
```ㅑ``` : 반복문 시작
```ㅕ``` : 현제 포인터의 값이 0이 될때까지 ```ㅑ```으로 가기
